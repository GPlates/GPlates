Welcome to the GPlates Python documentation!
============================================

*pygplates* is a Python module that provides access to `GPlates <http://www.gplates.org>`_
functionality for the `Python <http://www.python.org>`_ programming language.

This documentation includes an introduction to pygplates, how to set up pygplates,
coverage of the different areas of pygplates, cookbook examples, Frequently Asked Questions (FAQ) and
a detailed reference of the pygplates classes and functions.
